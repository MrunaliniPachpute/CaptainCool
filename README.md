# CaptainCool AI 🏏⚡

CaptainCool AI is a multi-agent IPL tactical strategist built using the Google Gemini ecosystem.

Instead of giving one generic chatbot response, the platform simulates a real IPL war-room where multiple AI agents debate live cricket decisions before producing a final captain-style tactical call.

---

## Features

- 🧠 Multi-agent AI orchestration
- 📊 Tactical debate + statistical validation
- 🔥 Risk analysis & counter strategies
- 👑 Final captain-style decision making
- 🎙️ IPL commentary-style explainability
- 🏏 Manual tactical simulation mode
- ⚡ Live Match Beta using CricAPI
- 🌌 Glassmorphism IPL broadcast UI

---

## 🤖 Multi-Agent Pipeline

### 🧠 Strategist Agent
Proposes tactical moves like bowling changes, field setups, and impact player usage.

### 📊 Stats Analyst Agent
Validates cricket logic using match context, venue conditions, and momentum.

### 🔥 Devil’s Advocate Agent
Challenges risky tactical decisions and creates strategic debate.

### 👑 Final Decision Agent
Combines all agent outputs into the final AI captaincy call.

### 🎙️ Commentary Agent
Transforms tactical reasoning into exciting IPL-style commentary.

# ⚙️ Setup

## Install

```bash
npm install
```

## Create `.env`

```env
GEMINI_API_KEY=your_key
CRICAPI_KEY=your_key
PORT=3000
```

## Run

```bash
npm start
```

Open:

```bash
http://localhost:3000
```
